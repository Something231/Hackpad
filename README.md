# Hackpad
Hack club starter project based on https://blueprint.hackclub.com/starter-projects/hackpad

This is just gonna be a circular devboard with a small dial in the centre for volume control.

Just making this to learn a bit more about doing circuit design mostly.

Repository includes:
- Kicad files in /PCB(kicad) 
- Raw Gerber in /gerber , also in CAD/gerber.zip
- Hackpad case 3d files in stl, step and f3d formats in /CAD
